# CodeAlpha Secure Coding Review

## Project Overview
This project was completed as part of the CodeAlpha Cyber Security Internship.

The objective of this task is to identify security vulnerabilities in insecure Python code and implement secure coding improvements.

---

## Technologies Used
- Python
- VS Code
- hashlib library

---

## Security Issues Identified
- Hardcoded credentials
- Weak passwords
- Plain text password storage
- No input validation
- No account lockout mechanism

---

## Security Improvements
- Password hashing using SHA-256
- Stronger password implementation
- Improved authentication process
